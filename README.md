# 🚀 Customer Churn Prediction

## 📌 Problem Statement
Customer churn is a major challenge for businesses. This project predicts whether a customer will leave the service based on behavioral and demographic features.

## 📊 Dataset
- Source: Kaggle
- Features: Age, Tenure, Support Calls, Payment Delay, Total Spend, etc.

## ⚙️ Workflow
1. Data Understanding
2. Data Cleaning (missing values, data types)
3. EDA (correlation analysis, insights)
4. Feature Engineering
5. Encoding & Scaling
6. Model Building
   - Logistic Regression
   - Random Forest
7. Evaluation
8. Feature Importance & Cross Validation

## 🤖 Models Used

### Logistic Regression
- Accuracy: ~89%

### Random Forest
- Accuracy: ~99%

## 📈 Key Insights
- 📞 Support Calls strongly impact churn
- 💰 High spending customers are less likely to churn
- ⏳ Payment delays increase churn probability

## 🧠 Feature Importance
Top features:
- Support Calls
- Total Spend
- Payment Delay
- Age
